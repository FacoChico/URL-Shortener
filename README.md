Для запуска приложения нужно собрать его, а затем запустить с параметром `-Djava.awt.headless=false`

```bash
./gradlew clean bootJar
```

```bash
java -Djava.awt.headless=false -jar build/libs/url-shortener-0.0.1.jar
```

Лимиты и TTL короткой ссылки считаются частью её неизменяемого контракта.
Изменение этих параметров для уже опубликованной ссылки нарушает предсказуемость поведения, усложняет конкурентную логику и аудит.
Изменение лимитов реализуется через удаление старой ссылки и создание новой, что упрощает модель, повышает безопасность и делает поведение системы прозрачным.
